# GammaLauncher
> The best launcher for Minecraft/Лучший лаунчер для Майнкрафт

## ✨ Features/Особенности

**English:**

- **Support for all versions of Minecraft** - from the old classics to the latest releases.
- **Quick launch** - everything is optimized for quick game launch.
- **User-friendly interface** - intuitive design with no unnecessary elements.
- **Mod management** - built-in support for popular mod loaders (Forge, Fabric, Quilt).
- **Customization options** - flexible customization of options.
- **Statistics** - convenient tracking of the number of launches, total game time, and the last session.
- **Auto-update** - always have the latest version of the launcher.
- **Multi-account** - work with multiple accounts. Create as many profiles as you want.
- **Lightweight** - minimal system resources for maximum game performance. Works even on a weak PC!
