# GammaLauncher
Лучший лаунчер для Майнкрафт

## ✨ Особенности

- **Поддержка всех версий Minecraft** — от старых классических до последних релизов.
- **Быстрый запуск** — все оптимизированно для быстрого запуска игры.
- **Удобный интерфейс** — интуитивно понятный дизайн без лишних элементов.
- **Управление модами** — встроенная поддержка популярных мод-лоадеров (Forge, Fabric, Quilt).
- **Настройка параметров** — гибкая настройка опций.
- **Статистика** — удобное отслеживание кол-ва запусков, общее время игры и последней сессии.
- **Автообновление** — всегда актуальная версия лаунчера.
- **Мультиаккаунтность** — работа с несколькими учётными записями. Создавайте сколько угодно профилей.
- **Лёгкий вес** — минимум ресурсов системы для максимальной производительности игры. Запустится даже на слабом ПК!
